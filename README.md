# My Capstone - Minimal AI Agent

## What this project contains
- `agent.py` : minimal interactive agent (CLI).
- `tools.py` : mock tools (turn_on_light, set_temperature, get_schedule).
- `memory.py` : simple file-backed memory.
- `evaluation/user_simulation_tests.json` : sample user simulation scenarios.
- `evaluation/eval_results.json` : (generated after running evaluation).
